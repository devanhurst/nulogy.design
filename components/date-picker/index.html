Date Picker

Date pickers allow users to easily enter dates.

import { DatePicker } from "@nulogy/components";

<DatePicker
  selected={new Date("Fri, 01 Jan 2019")}
  dateFormat="MMMM d, yyyy"
  onChange={(val) => val}
  onInputChange={(val) => val}
/>
